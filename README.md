# arizona_core

This is a breakout of the core modules from [Arizona Framework](https://github.com/arizona-framework/arizona). All credit goes to the original maintainers and contributors of the Arizona project — this wouldn't exist without their work.

This is an experiment in separating the core template engine, parser, renderer, and diffing logic so it can be used independently with [Nova](https://github.com/novaframework/nova) and [Cowboy](https://github.com/ninenines/cowboy) without pulling in the full Arizona framework.
